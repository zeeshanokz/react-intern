# ARCA Store - Cool Keyboard Shop

ARCA is a simple online store for fancy mechanical keyboards, custom keycaps, and desk stuff. It's made with just Tailwind CSS and React, no extra UI libraries.

## Why I Built It This Way

- **What to Sell:** I picked keyboards because they look awesome with dark themes and detailed photos.
- **Looks:** Used "Outfit" font from Google Fonts and zinc colors with cyan for buttons. Keeps it modern and techy.
- **Keeping It Simple:** Cart stuff is handled in the main App component and saved to local storage. No big libraries needed.

## How to Run

Just do:

```sh
npm install
npm run dev
```

(Use Tailwind CSS v3.x)

Have fun with the store!
